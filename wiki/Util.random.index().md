randomly chooses an index from an array or object<br>
type: `Function`<br><br>
arguments: 
- object `Array/Object`
<br><br>

```js
util.random.index([ "a", "b", "c" ]); // 2

util.random.index({ key1: "val1", key2: "val2", key3: "val3" }); // 1
```

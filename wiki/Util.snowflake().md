turns a date into a snowflake<br>
type: `Function`<br><br>
arguments:
- date `Date`

<br>

```js
let dt = new Date("2018-11-28T09:01:00.000Z"); // November 28, 2018

util.snowflake(dt); // 517263631319040000
```
